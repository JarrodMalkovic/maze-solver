## Run Locally

### Clone

- Clone this repo to your local machine using `https://github.com/jarrodmalkovic/maze-solver.git`

### Install dependencies

```bash
npm install
```

### Start server

```bash
npm start
```

- Then open the project in a browser on http://127.0.0.1:8080
